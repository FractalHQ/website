# Routify Full Template

# Requirements
```
12 >= NodeJS <= 14
NPM >= 6
```

# Install:
```
npx degit Oxova/routify project-name
cd project-name
npm install
```

# Commands
- Dev
    ```bash
    npm run dev
    ```

- Build (Static)
    ```bash
    npm run build
    ```

- Start
    ```bash
    npm run start
    ```

# Features
- Svelte & Routify
- SCSS
- Auth boilerplate
- Favicon
- Font awesome support (uncomment line in __app.html)
- Robots.txt
- Autoprefixer
- Postcss import
- Nollup
- Rollup
- SSR & Static
